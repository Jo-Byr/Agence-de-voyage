Projet : Site Web d'une agence de voyage
CPE Lyon, 3ETI, 2020-2021
Ankou Pierre-Olivier, Bouyer Jonathan


La gestion du zoom est imparfaite pour la plupart des pages et aucune n'est adaptée à une lecture sur écran de téléphone portable.
L'affichage des destinations sur la page d'accueil est sujet à une certaine lenteur (environ 0.5 secondes).
La carte disponible sur le site (HTML/Carte.html) est une version modifée de celle du site Simplemaps.com (les copyrights ont été gardés dans le fichier svg).
Comme précisé en bas de la page d'accueil, l'adresse mail fournie sur le site est fictive, mais le numéro de téléphone ne l'est pas.
La page A propos laisse sous-entendre une présence de l'agence sur les réseaux sociaux et les logos sont présentés comme des liens, cependant nous n'avons pas créé ces pages donc les images ne sont pas réellement des liens.
La page "Panier" n'a pas pu être faite et est donc toujours vide.
Les input date présents sur différentes pages donnent lieu à un avertissement du W3C Validator, cependant cela n'impactant pas le fonctionnement du site, nous avons laissé les input tel quels.
